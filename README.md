# Iris Flower Classification

## Objective
Build a Machine Learning model to classify Iris flowers into three species based on sepal and petal measurements.

## Algorithm Used
- Random Forest Classifier

## Libraries Used
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Dataset
- IRIS.csv

## Features Used
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

## Steps Performed
1. Loaded the dataset.
2. Explored the data using head(), info(), and describe().
3. Checked for missing values.
4. Encoded the species column using LabelEncoder.
5. Split the dataset into training and testing sets.
6. Trained the model using Random Forest Classifier.
7. Predicted the flower species.
8. Evaluated the model using Accuracy Score, Classification Report, and Confusion Matrix.

## Results
- Accuracy: 100%
- Successfully classified Iris flowers using Machine Learning.
